# AAiOS CLI Reference Manual
## Version 5.3.2 — Command Line Options

This document documents all top-level CLI commands registered on the `aaios` executable.

---

### 1. Diagnostic Commands
*   `aaios doctor`: Run system checks and apply self-healing.
*   `aaios certify`: Execute production validation and print compliance cards.
*   `aaios benchmark`: Measure cold boot speeds and system latency.

### 2. Operation Commands
*   `aaios backup`: Create, list, delete, or restore backups.
*   `aaios reset`: Restore configurations or wipe databases.
*   `aaios cleanup`: Delete temporary files and caches.
*   `aaios monitor`: Run continuous health telemetry monitoring.
