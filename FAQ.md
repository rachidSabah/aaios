# AAiOS Frequently Asked Questions (FAQ)
## Version 5.3.2 — FAQ Reference

This document answers common questions about AAiOS.

---

### 1. General Questions

#### Q: How does AAiOS differ from other agent frameworks?
AAiOS acts as an Agentic Operating System, abstracting agent libraries behind a generic runtime and supervisor.

#### Q: Can I run AAiOS offline?
Yes, using Ollama or LM Studio, you can run LLMs and execute agents locally.
