# AAiOS v5.3.2 Release Notes
## Version 5.3.2 — Enterprise Release Reference

This document describes new features, fixes, and updates included in AAiOS v5.3.2.

---

### 1. Release Summary

AAiOS v5.3.2 is an enterprise-grade release introducing diagnostic, validation, self-healing, recovery, update, and packaging services.

### 2. Key Additions

*   **Doctor / Self-Healing**: Diagnostic checking and repair systems.
*   **Backup / Restore**: Encrypted backup archives with pre-restore checks.
*   **Update Manager**: Distribution channel routing and migration support.
*   **Validator & Certify**: Compliance testing and compliance certificates.
*   **Packaging**: Release artifact creation with SBOM and manifests.
